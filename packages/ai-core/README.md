# @xake/ai-core

Prompt templates, tool schemas, guardrails, and model routing for the XAKE AI assistant.

Stage 8 wires Claude Sonnet 4.6 as the default interactive model, Haiku 4.5 for lightweight tasks, and Opus 4.7 behind admin/premium flows. Prompt caching is enabled for system instructions, tool schemas, and long workspace context. The assistant can analyse, explain, summarise, screen, and draft — it never places trades on its own.

# @xake/web

Next.js App Router frontend for XAKE.

## Run

```bash
pnpm --filter @xake/web dev
```

Open `http://localhost:3000`.

## Stage 0 scope

Minimal shell only. Renders the XAKE wordmark, a paper-environment badge, and a checklist confirming the scaffold. No routing, no auth, no data. Stage 2 introduces the design system; Stage 3 wires Clerk; Stage 4 subscribes to the mock feed.
